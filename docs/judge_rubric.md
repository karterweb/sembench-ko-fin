# Judge Rubric

SPS is scored from 0 to 20.

| Dimension | 0 | 2 | 4 |
|---|---|---|---|
| Intent Preservation | Wrong task | Partially answers | Same task |
| Fact Accuracy | Contradicts facts | Some imprecision | Core facts correct |
| Reasoning Completeness | Missing reasoning | Partial reasoning | Full reasoning preserved |
| Actionability | Misleading | Some next steps | Same action possible |
| Hallucination Absence | Invented facts | Minor unsupported detail | No hallucination |

Korean examples and calibration notes live in `sembench/judge/prompts/rubric_ko.md` and `docs/judge_calibration.md`.
